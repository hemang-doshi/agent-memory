# Live-Agent Evaluation Report

The reproducible report is generated by:

```bash
agentmem eval live --write-report --json
```

The checked-in harness covers:

- avoid `npm install` in a `pnpm` repo;
- avoid fragile file;
- avoid known failed approach;
- respect architecture decision;
- respect command preflight;
- propose reusable learning;
- ignore stale/superseded memory;
- avoid secret-bearing memory.

This local harness does not claim that external agents always obey memory or always produce better code.
