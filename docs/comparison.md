# Comparison

Agent Memory is a local CLI and MCP memory layer. It does not replace a hosted vector database, cloud sync service, or agent runtime.

Compared with plain project docs, Agent Memory adds structured memory records, retrieval explanations, command preflight, protocol receipts, candidate review, safety scanning, lifecycle operations, backups, and reproducible local evals.

Compared with external memory services, Agent Memory defaults to local SQLite and local indexes. External providers are opt-in only.
