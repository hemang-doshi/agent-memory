# Examples

Examples are small, local workflows that can be copied into temporary projects while testing Agent Memory behavior.

- [quickstart](./quickstart/README.md): initialize a local project, add memory, retrieve a packet, preflight a command, propose a candidate, and read a receipt.
- [avoid-repeated-mistake](./avoid-repeated-mistake/README.md): save a repo-specific package-manager mistake, inject it before a new task, and preflight the risky command.
